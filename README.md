# Reto FOODMAP

## Objetivo
Crea una web-app que a través de un input pueda filtrar los restaurantes que se encuentran cerca de ti.
## Herramientas

* HTML
* CSS
* Framework bootstrap
* Jquery

##filtros
El tema que da a en esta aplicación es sobre postres y el filtro es de la siguiente manera:
-  helados
- cupcakes
-  sanguito
-  chocolate
-  suspiro
-  mazamorras
